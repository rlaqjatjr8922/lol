## ⚠️ 한계 및 개선 예정

본 프로젝트는 챔피언 선택 단계의 전체 자동화를 목표로 하였으나,  
실시간 이미지 처리 과정에서 발생하는 성능 문제로 인해 일부 기능은 최종 구현 단계에서 중단되었습니다.

특히:

- 다수 ROI 영역에 대한 반복적인 템플릿 매칭 연산
- 프레임 단위 실시간 처리 구조
- UI 렌더링과 분석 로직의 동시 실행

이로 인해 CPU 사용량이 급격히 증가하며  
일부 환경에서 프로그램이 비정상적으로 정지하는 문제가 발생했습니다.

이에 따라 현재는 안정적인 범위 내에서:

- 밴 감지
- 픽 감지
- GPT 기반 추천 시스템

까지 구현된 상태이며,

추후에는 다음과 같은 방향으로 개선을 고려하고 있습니다:

- ROI 연산 병렬 처리
- 템플릿 매칭 최적화
